# 🚀 Bash & Python Automation Playbook for DevOps

A collection of real-world automation scripts used in DevOps, SRE, and Platform Engineering.

This repository focuses on solving production problems using:
- Bash scripting for system-level automation
- Python for cloud, APIs, and intelligent workflows

## 🎯 Goal
Automate repetitive operational tasks, reduce manual effort, and improve reliability in production systems.

## 🔥 Key Areas
- Linux automation
- AWS automation
- Kubernetes operations
- Monitoring & alerting
- CI/CD scripting

## 📂 Structure
- bash/ → system automation scripts
- python/ → cloud + API automation
- kubernetes/ → cluster operations
- ci-cd/ → deployment scripts

## 🧠 Real-World Focus
Each script includes:
- Use Case
- Commands / Code
- Explanation
- Where it's used in production
- Impact
